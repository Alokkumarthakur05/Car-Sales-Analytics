# Car-Sales-Analytics
Developed an interactive Sales Analytics dashboard using Microsoft Excel to analyze car sales data from a 13-year Kaggle dataset containing seven key attributes. Performed complete data cleaning to ensure accuracy and consistency, enabling reliable analysis and insights. Conducted in-depth exploratory analysis to identify five high-impact business insights related to sales trends and performance. Created advanced pivot tables and dynamic pivot charts to visualize patterns, correlations, and long-term trends. Integrated slicers to enable real-time interactivity, improving data exploration and supporting data-driven decision-making. Leveraged strong analytical thinking and visualization techniques to present insights in a clear and actionable manner.
